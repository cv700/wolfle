How to Play

The Emoji Werewolf Game is a memory and matching game with a twist:

  • Click on a tile to reveal an emoji.
  • Find matching pairs of emojis by remembering their locations.
  • Double-click the werewolf emoji to win the game!
  • If you double-click the wrong tile, the game ends.
  • Matched emojis show arrows pointing towards the werewolf.

Try to match all the pairs and find the werewolf before time runs out!
